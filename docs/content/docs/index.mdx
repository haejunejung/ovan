**hoyst** is an overlay library for **modals, toasts, and dialogs** on **React** (web) and **React Native**.

It provides Slot-based context preservation, close/unmount separation, Promise-based result returns, and fundamentally solves the native layer stacking issues in React Native.

## Installation

### React (web)

```bash
pnpm add hoyst react react-dom
```

### React Native

```bash
pnpm add hoyst react react-native react-native-teleport
```

## Quick Setup

Wrap your app with `OverlayProvider` and place an `OverlaySlot`:

```tsx title="App.tsx"
import { OverlayProvider, OverlaySlot } from "hoyst"

function App() {
  return (
    <OverlayProvider>
      <OverlaySlot />
      {/* your app content */}
    </OverlayProvider>
  )
}
```

Then open overlays from anywhere:

```tsx
import { overlay } from "hoyst"

overlay.open(MyModal, { overlayId: "my-modal" })
```

No setup file needed — just import and use.

<Callout type="info">
  Need multiple isolated overlay trees? Use `createOverlayProvider()` to create
  separate instances. See the [API
  Reference](/docs/api#createoverlayprovideradapter) for details.
</Callout>
